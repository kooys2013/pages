치중진담 (治中眞談)이란?

치료에 대한 진실된 이야기. Team CareFlow (장회영 신경외과·남유미 마취통증의학과·물리치료사 김실장)가 함께 만드는 기능재활 채널. 남스어프로치와 같은 기능적 접근법을 공유함.

핵심 이론

🐢
거북목 — 턱 당기기(Chin Tuck) 하지 마세요
6분 · 남유미 원장
오해 교정거북목
  • 거북목은 목만의 문제가 아님. 전신 기능 저하의 결과
  • 턱 당기기를 열심히 해도 뒷목이 더 아파지는 이유 → 근본 원인을 건드리지 않기 때문
  • 먼저 호흡근·코어 기능을 회복해야 목 근육 과사용이 줄어든다
💪
기능회복운동 — 왜 세게 하면 안 되는가?
23분 · Team CareFlow
핵심 원칙
  • Henneman의 크기 원리(Size Principle): 낮은 부하 → Type 1(안정화) 먼저 동원, 높은 부하 → Type 2(이동성)만 씀
  • 세게 하면 Type 2 섬유만 일하고 Type 1은 쉰다 → 재활 목적 달성 못함
  • 낮은 부하로 Type 1을 충분히 활성화한 후 점진적으로 올려야
🔄
반사적 안정화 (Reflexive Stabilization)
28분 · Team CareFlow
이론
  • 재활 훈련 중 자동으로 안정화 패턴이 만들어지게 하는 전략
  • 의식적으로 특정 근육을 조이는 것이 아니라 자연스러운 반사 반응을 유도
  • 훈련이 쌓이면 일상 동작에서 자동으로 적용됨
🦴
목·어깨 통증 — 허리·골반도 함께 치료해야 하는 이유
14분 · 장회영 원장
전신 연결
  • 목어깨 통증 환자의 허리·골반 기능 저하가 함께 존재하는 경우가 많음
  • 전신 연결 고리: 골반 불안정 → 코어 약화 → 보조호흡근 과사용 → 목·어깨 과부하
  • 목만 치료하면 재발. 전신 기능 회복이 필수
😣
코어 운동 열심히 했는데 더 아픈 이유
20분 · 남유미 원장
흔한 실수
  • 잘못된 코어 운동 = Type 2 섬유만 자극 → 오히려 근육 긴장도 상승
  • 복부를 억지로 조이거나 힘을 주면 국소 코어(Type 1)가 아닌 외부 근육이 반응
  • 낮은 부하 + 느린 속도 + 감각 인지가 먼저
💉
주사만 맞고 운동 안 하면 100% 재발
10분 · 장회영 원장
  • 주사(신경차단술)는 통증을 줄여주지만 기능 저하는 해결 못함
  • "통증이 줄어든 기회"를 운동으로 활용해야 재발 방지
  • 주사 → 기능회복 운동 → 일상 관리 순서가 올바른 치료 흐름
🎯
디스크 때문이라고 굳게 믿게 되는 환경
10분 · 장회영 원장
  • MRI에서 디스크가 보인다고 그게 통증의 원인이라는 보장이 없음
  • 디스크 돌출 = 통증의 원인이라는 인식이 과잉 치료로 이어짐
  • 기능적 접근으로 디스크 문제 없이도 회복 가능한 경우가 많다

기능재활 CareFlow 기본 개념

🏎️
CareFlow 삼두마차 — 환자·치료사·의사의 3인 경주
Team CareFlow
  • 기능재활은 환자·치료사·의사 셋이 함께 해야 효과
  • 환자가 운동을 이해하고 집에서 실천해야 치료가 완성됨
  • 치료받는 시간(30분)보다 나머지 23시간 30분이 더 중요
✅
운동 순서

마사지(이완) → 복강내압 훈련 → 전거근 안정화 → 클램쉘(대둔근) → 장요근 — 낮은 부하부터, 느리게, 감각을 찾으면서.

Step 0 — 이완 마사지

⓪ 준비

마사지볼로 이완하기

운동 전 딱딱한 근육을 풀어야 Type 1 섬유가 활성화됨. 허벅지→종아리→골반 순서.

Step 1 — 복강내압(IAP) 훈련

① 기초

3개월 아기 자세(Three-Month Position)에서 복강내압 훈련

등을 바닥에 붙이고 엉덩이 90°, 무릎 90° 자세에서 균형 잡기.

1
자세 셋업

등을 바닥에, 고관절·무릎 각도 90°. 발목은 자연스럽게.

2
배 압력 살짝 만들기

억지로 배를 조이는 게 아님. 숨을 멈추지 않고 자연스럽게 팽팽한 통 느낌.

3
균형 유지

이 자세에서 흔들리지 않게 1분 버팀. 골반이 기울면 부하가 너무 높은 것.

❌
잘못된 방법

배를 최대로 꽉 조이기, 숨 참기, 배를 일부러 홀쭉하게 당기기 — 전부 Type 2 자극.

Step 2 — 전거근(날개뼈) 안정화

② 기초 · 목디스크 핵심

앉아서 전거근 훈련 — 어깨 강제로 내리지 마세요

날개뼈를 제자리에 고정하는 전거근을 활성화. 목·어깨 통증의 근본 원인 해결.

1
앉아서 자세 잡기

의자에 앉아 등이 흐트러지지 않게. 어깨를 억지로 내리거나 뒤로 당기지 않음.

2
날개뼈 살짝 앞으로

벽을 손바닥으로 살짝 밀듯이. 강하게 밀면 Type 2 동원.

3
겨드랑이 아래 느낌 기다리기

1분 후 겨드랑이 옆구리 아래쪽에서 당기는 느낌 = 전거근 활성화 성공.

⚠️
흔한 실수

어깨를 강제로 아래로 내리면 전거근이 아닌 다른 근육이 일함. 어깨 위치는 자연스럽게.

Step 3 — 클램쉘 (대둔근 하부)

③ 중급 · 골반 안정화

클램쉘 + 국소 코어 세팅

대둔근 하부 섬유를 활성화해서 골반 안정성을 높이는 운동. 허리·목 통증 전신 연결 핵심.

1
옆으로 눕기 + 코어 세팅

국소 코어 느낌 먼저 잡고 시작. 코어 없이 하면 골반이 흔들림.

2
무릎 조개처럼 천천히 벌리기

빠르게 하면 Type 2 동원. 2~3초에 걸쳐 천천히.

3
골반이 뒤로 돌면 실패

골반 회전 = 대둔근이 아닌 다른 근육이 대신함. 범위를 줄여서 다시.

Step 4 — 장요근 훈련

④ 중급

장요근(Iliopsoas) 국소 코어 훈련

통증 없이 장요근을 자극하는 저부하 접근법.

CareFlow 핵심 원칙

1
통증 부위 ≠ 원인 부위

목이 아파도 원인은 골반·코어일 수 있다. MRI에 보이는 디스크가 통증 원인이 아닐 수 있다.

2
낮은 부하로 Type 1 먼저

Henneman 원리: 낮은 부하에서 Type 1(안정화) 먼저 동원. 세게 하면 Type 2만 쓴다.

3
주사 + 운동 = 세트

주사로 통증을 줄인 기회에 기능 회복 운동을 해야 재발 방지. 주사만으로는 100% 재발.

4
수술 전 비수술 먼저 상담

수술이 필요한 경우도 있지만, 수술 전 기능적 접근 여부를 확인해야 한다.

전신 연결 이해

목·어깨 통증의 전신 연결 고리
발·종아리 경직→ 골반 불안정→ 코어 약화→ 횡격막 기능 저하→ 보조호흡근 과사용→ 목·어깨 통증

오해 vs 올바른 접근

❌ 오해
  • 거북목 → 턱 당기기로 교정
  • 코어 → 세게 조여야 효과
  • 디스크 보이면 → 디스크 수술
  • 주사 맞으면 → 다 나음
  • 통증 부위 = 원인 부위
✅ 올바른 접근
  • 전신 기능 회복이 먼저
  • 낮은 부하로 Type 1 활성화
  • 기능 저하가 진짜 원인인지 확인
  • 주사 기회에 운동 병행
  • 전신 연결 고리로 원인 추적

핵심 용어 사전

Henneman의 크기 원리 (Size Principle)

근육이 수축할 때 작은(Type 1) 운동단위부터 동원되고, 부하가 커질수록 큰(Type 2) 운동단위가 추가됨. 낮은 부하에서만 Type 1(안정화 근육)이 주로 일한다는 원리. 재활 운동에서 낮은 부하가 필요한 과학적 근거.

전구처럼 약한 전류(낮은 부하)에서는 소형 전구(Type 1)만 켜지고, 강한 전류(높은 부하)에서는 대형 전구(Type 2)가 추가로 켜진다.
Type 1 근섬유 (안정화 / 지근)

자세 유지, 지구력 담당. 낮은 강도(최대근력 25% 미만)에서 활성화. 근비대 없이 활성화 능력이 목표. 재활 초기 훈련의 주 대상. 천천히, 낮게, 오래.

Type 2 근섬유 (이동성 / 속근)

순발력, 폭발력 담당. 높은 강도에서 동원. 근비대 O. 재활 초기에는 Type 2가 일하면 실패. 헬스장에서 키우는 근육.

반사적 안정화 (Reflexive Stabilization)

의식적으로 "이 근육 조여야지"가 아니라 움직임 중에 자동으로 안정화 패턴이 발동되는 상태. 재활이 성공하면 일상 동작에서 자동으로 적용됨.

처음 자전거를 탈 때는 의식적으로 균형을 잡지만, 익숙해지면 자동으로 잡히는 것처럼.
전거근 (Serratus Anterior)

겨드랑이 옆구리 근육. 날개뼈를 흉곽에 고정. 이 근육이 약하면 날개뼈가 떠서 팔·목으로 과부하 전달. 전거근 운동 성공 시 겨드랑이 아래가 당기는 느낌이 남.

복강내압 (IAP)

배 안의 압력. 코어 근육들이 만드는 내부 지지력. 허리·척추의 쿠션 역할. 억지로 조이는 것이 아니라 자연스럽게 팽팽한 통 느낌이 올바른 방법.

신경차단술 (주사치료)

통증 신호를 차단하는 주사. 통증은 줄지만 기능 저하는 해결 안 됨. 주사 후 기능회복 운동을 반드시 해야 재발 방지. 주사만 반복하면 결국 재발.

이해 확인 퀴즈

1
거북목에 턱 당기기(Chin Tuck)가 효과가 없는 이유는?
전신 기능 문제가 근본 원인이기 때문. 목 근육의 과사용은 횡격막 약화로 인한 보조호흡근 과사용에서 비롯됨. 턱만 당기면 이 연결 고리를 해결하지 못하고 오히려 이미 피로한 목 근육을 더 혹사시킴.
2
코어 운동을 열심히 했는데 왜 더 아플까?
Type 2 섬유만 자극했기 때문. 세게 또는 빠르게 운동하면 Type 1(안정화)이 아닌 Type 2(이동성)가 동원됨. 이 상태에서는 근육 긴장도만 높아지고 안정화 기능은 향상되지 않아 오히려 통증이 증가할 수 있음.
3
전거근 운동 시 어깨를 강제로 내리면 안 되는 이유는?
다른 근육이 대신 일하기 때문. 어깨를 억지로 내리면 승모근 하부나 다른 근육이 대신 과활성화됨. 전거근은 자연스러운 자세에서 날개뼈가 앞으로 살짝 이동하는 느낌으로 찾아야 한다.
4
주사치료를 받은 후 꼭 운동을 해야 하는 이유는?
주사는 통증만 줄이고 기능 저하는 해결 못하기 때문. 기능 저하가 남아있으면 시간이 지나 재발함. 주사로 통증이 줄어든 "기회의 창"을 이용해 기능회복 운동을 해야 비로소 재발 방지가 됨.
